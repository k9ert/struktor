Traditionell lernt man das Programmieren mit einem "Hallo Welt"-Programm. Dieses tut nichts anderes als den Text "Hallo Welt!" auf dem Bildschirm auszugeben. Das ist nur ein Befehl (wenn man nicht in Maschinensprache bzw. Assembler programmiert). In Struktogrammen wird ein Befehl als ein Rechteck dargestellt.
Unten sehen Sie (Bitte etwas Geduld) nun dieses einfache Programm. Sie können es ausführen indem Sie auf den "execute-Button" klicken (bitte nochmal etwas Geduld).

In diesem einfachen Beispiel werden keine Variablen verwendet. Dafür gibt dieses Kommando etwas aus (Output). Alle Commands können grundsätzlich eingeteilt werden in:
    Input (Eingabe)
    Calculation (Verarbeitung)
    Output (Ausgabe)
Aus den Anfangsbuchstaben leitet sich der Name für ein wichtiges Prinzip in der Programmierung ab. Das EVA-Prinzip. Es besagt das man immer zuerst alle Daten sammeln sollte (Input). Danach erst die Berechnung (Calculation) durchführt und dann das Ergebnis ausgibt (Output).

Das obige Beispiel können Sie nur ausführen. Im weiteren Verlauf der Einführung, werden Ihnen in den Beispielen immer mehr Möglichkeiten zur Verfügung gestellt. Der Vorteil ist, daß durch die geringere Komplexität die Aufmerksamkeit auf das wesentliche gelenkt werden kann.
