In dieser Lektion soll gezeigt werden, wie Sie Struktogramme speichern bzw. Laden können.
Versuchen Sie zunächst das Struktogramm der letzten Seite aus den Kopf neu zu erstellen.
Haben Sie Ihr Programm erstellt ? Dann klicken Sie nun auf "Save". Im folgenden Dialog aktivieren Sie main und klicken "Ok". Es springt ein Fenster auf in dem der Code des Struktogramms sichtbar ist. Markieren Sie mit der Maus den kompletten Code (Achten Sie darauf, daß Sie wirklich alles markiern). Danach drücken sie Ctrl-C (oder auch Strg-C). Sie haben nun den Code in die Zwischenablage kopiert. Sie können nun über "Start/Zubehör/Editor" einen Editor startten und hier den Code über Ctrl-V einfügen und anschließend abspeichern.
Wollen Sie ein Struktogramm wieder laden, funktioniert dies genau andersherum. Klicken Sie im unteren Struktor auf "load". Fügen Sie den Code über Ctrl-V ein und klicken Sie auf Ok. Es erscheint wieder das ursprüngliche Struktogramm.
