Das folgende Programm fragt Sie nach Ihrem Namen, errechnet danach einen Begrüßungstext und gibt diesen danach aus. Insgesamt besteht das Programm aus 3 Kommandos. Es ist ein sehr einfaches Beispiel für das EVA-Prinzip.

Im Gegensatz zum letzten "Hallo-Welt"-Programm benutzt dieses Programm eine Variable, in der der Name gespeichert wird, und in dem im zweiten Schritt der Begrüßungstext gespeichert wird. Dies geschieht durch die Zuweisung:
name = "Hallo "+name+" !"; (Fast jedes Kommando muß mit einem Semikolon abgeschlossen werden)
Jede Variable die verwendet wird muß vorher deklariert werden. Sie können sich die Deklarationen in diesem Programm im "Declaration-View" anschauen (Pull-Down-Menü). Die Variable "name" ist eine String Variable. D.h. es können in dieser Variable Zeichenketten gespeichert werden. Verändern Sie den Typ oder den Namen der Variable, erhalten Sie seltsame Fehlermeldungen. Probieren Sie es aus !
