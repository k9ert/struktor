Die Beiden vorherigen Programme waren recht starr. Innerhalb des Programms konnte nicht entschieden werden, ob ebtl. das Programm einen anderen Verlauf nehmen soll. Mit der sogenannten Verzweigung (Condition) ist dies auf einfache Art möglich
Wie Sie sehen, wird das Programm nicht ganz korrekt dargestellt (Ein Fehler in Struktor). Das können Sie selber beheben. Bewegen Sie den Mauszeiger an den rechten Rand des ersten Elements. Klicken Sie auf den Rand und halten Sie die Maustaste gedrückt. Nun können Sie die Größe verändern.
Auf die gleiche Art können Sie die Position auf dem Bildschirm verändern indem Sie das schwarze Rechteck des ersten Elements klicken. Auch das Seitenverhältnis der Verzweigung kann auf die gleiche Art und Weise verändert werden. Die grafische Representation der Elemente hat aber keinen Einfluß auf die Ausführung.
Auch dieses Programm benützt Variablen. Auch alle weiteren Programme werden Variablen verwenden. Variablen sind das Salz in der Suppe der Programmierung. Ohne Variablen wäre kaum ein vernünftiges Programm möglich.
